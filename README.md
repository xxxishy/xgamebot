<details><summary><h3>How Does xGame work?</h3></summary>

xGame is not like other bots. Unlike other bots that have a dashboard, xGame runs on your PC. We will give you all the scripts for you to paste into a code software (e.g Visual Studio Code). The bot will the run on your PC, not ours. But if you follow everything in the instructions correctly, it should be the exact same.
</details>
